ATS Resume Scanner & Scoring System

This project is designed to simulate an Applicant Tracking System (ATS) used by recruiters to filter resumes. It analyzes resumes by extracting relevant keywords and comparing them with job descriptions.

## 🚀 Features
- Resume parsing (PDF/Text)
- Keyword extraction using NLP
- Job description matching
- ATS compatibility scoring
- Skill gap analysis

## 🎯 Use Case
- Helps job seekers optimize resumes
- Improves chances of passing ATS filters
- Identifies missing skills based on job descriptions
